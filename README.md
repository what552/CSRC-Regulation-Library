# CSRC Regulation Library

中国证券监管法规库 — 面向资本市场从业人员的结构化法规知识库，收录证监会、三大交易所及相关自律组织的核心监管文件。

A structured regulatory knowledge base covering China's securities market, including regulations from the CSRC, stock exchanges (SSE, SZSE, BSE), and self-regulatory organizations.

## 目录结构

```
├── 000-目录/           索引与导航
├── 001-附件/           附件（PDF 指引、截图、操作手册等）
├── 100-股票发行审核/    IPO、再融资、创业板、科创板、北交所
├── 200-债券发行审核/    公司债、企业债、可转债、可交债、ABS
├── 300-证券发行信息披露/ 招股说明书、法律意见书等披露准则
├── 400-证券发行保荐/    保荐管理办法、尽职调查工作准则
├── 500-询价与承销/      发行定价、网上网下发行、投资者管理
├── 600-证券上市与交易/   上市规则、交易规则、融资融券
├── 700-持续督导/        公司治理、定期报告、股权激励、减持回购

└── 900-财务顾问/        并购重组、收购、分拆上市、反垄断审查
```

## 监管主体覆盖

- **中国证券监督管理委员会（CSRC）** — 上位法规与部门规章
- **上海证券交易所（SSE）** — 主板 + 科创板规则
- **深圳证券交易所（SZSE）** — 主板 + 创业板规则
- **北京证券交易所（BSE）** — 北交所全套业务规则
- **全国股转系统（NEEQ）** — 新三板挂牌、定向发行、分层管理
- **中国证券业协会（SAC）** — 承销、网下投资者管理等自律规则

## 使用方式

本库以 Markdown 格式组织，推荐使用 [Obsidian](https://obsidian.md) 打开：

1. 克隆仓库
   ```bash
   git clone https://github.com/what552/CSRC-Regulation-Library.git
   ```
2. 在 Obsidian 中选择「打开文件夹作为仓库」，指向克隆目录
3. 从 `000-目录/0.证券监管法规总目录.md` 开始浏览

每个法规文件包含 YAML front-matter 元数据：

```yaml
---
发布主体: 中国证券监督管理委员会
发布日期: 2023-02-17
状态: 现行有效
tags:
  - 证监会/IPO
  - 保代考试/绝对核心法规
---
```

可配合 Obsidian Dataview 插件按发布主体、日期、标签等维度进行筛选和检索。
